# Numpy - Go/Python Starter Kit

Example project for Go-side writer and Python-side reader for
Numpy's `.npz` file format.

## Prerequisites

Make sure to run

```bash
./setup.sh
```

## Go-side writer

```bash
go run *.go
```

## Python-side reader

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```
